D3-iOS-app
===
A iPhone app which is Based on SAE and iOS platform。

功能
---

1. 基于SAE平台的，抓取信息的WEB端；包括抓取服务器状态，官方蓝贴等官方信息；
2. 大菠萝各种属性计算器，暂时未定具体内容；
3. 一个iOS客户端，可以浏览最终结果；


技术
---
1. Mac-OS
2. Mysql, Memcached
3. Flask
4. python, html, objective-C
